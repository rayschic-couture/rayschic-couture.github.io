Guide Rapide d'Utilisation

  • Cliquez sur "SE CONNECTER" et utilisez vos identifiants envoyés par email
  • Sélectionnez une collection (Costumes, Chemises, etc.) dans le menu de gauche
  • Choisissez l'image à modifier (Costume 1, Costume 2, etc.)
  • Cliquez sur "Choose an image" pour sélectionner la nouvelle photo
  • Sauvegardez avec "Save" - l'image est automatiquement remplacée sur le site
  • Attendez 1 minute et rafraîchissez le site pour voir le changement
